# william

Personal Portfolios Website of William Wei Ming's

## Prerequisites

```yaml
node.js: 20.20.2
npm: 10.8.2

next.js: 16.0.10
react: 19.2.0
```

In `.env.local` file:

```env
NEXT_PUBLIC_WEB3FORMS_KEY=
```

## Getting Started

```bash
npm i

npm run dev

npm run build

npm start
```

Visit http://localhost:3000/
